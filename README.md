# BMI Calculator 
## Team Name - CODE_SOURCE 
A responsive web application that calculates Body Mass Index (BMI) with health categorization. Built with HTML5, CSS3, and C for backend logic.

- ## Features
- **Accurate BMI Calculation**  
  `BMI = weight (kg) / (height (m))²`
- **Instant Health Classification**:
  - Underweight (BMI < 18.5)
  - Normal (18.5 ≤ BMI < 25)
  - Overweight (25 ≤ BMI < 30)
  - Obese (BMI ≥ 30)

 ## Installation
    
### Prerequisites
- GCC compiler (for C program)
- Modern web browser

  ## Usage
1) Enter height (cm) and weight (kg)
2) Click "Calculate BMI"
3) View results with health category
